## COCO Dataset
- data format for machine learning
- for
  - object detection
  - segmentation
  - keypoint detection. 

- object detection paper (for computer vision) COCO 2017
- 많은 라이브러리들이 COCO Dataset 으로 사전 학습된 (pre-trained) 모델 제공. 

  - training dataset : 118,000
  - validation dataset: 5,000
  - test : 41,000
